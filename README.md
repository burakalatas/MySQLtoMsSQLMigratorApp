## Mysql to Mssql Migrator app

This app can help you while migrating from mysql to mssql. 
Unlike other migration tools, this tool don't throw uniqueidentifier error while adding guid to uniqueidentifier column.
You don't have to select anything specifically. The app will match tables which has same name and which has same column names.
If there is an unmatched columns it will throw an error, and you can ignore it and fix it later than you can move it.
